🎯 VOS 3 ACTIONS (10 minutes total)

ACTION 1: Initialiser Supabase (5 min)
┌────────────────────────────────────────────────────────────────┐
│ 1. https://app.supabase.com → SQL Editor                       │
│ 2. Copier: scripts/sql/ndjobi-init-database.sql               │
│ 3. Exécuter (RUN)                                              │
│ 4. Attendre: ✅ Base de données NDJOBI initialisée             │
└────────────────────────────────────────────────────────────────┘

ACTION 2: Configurer .env.local (2 min)
┌────────────────────────────────────────────────────────────────┐
│ 1. Supabase → Settings → API                                   │
│ 2. Copier "service_role key" (⚠️ PAS "anon")                   │
│ 3. Créer .env.local à la racine                                │
│ 4. Ajouter la clé (voir CONFIGURATION-ENV.md)                  │
└────────────────────────────────────────────────────────────────┘

ACTION 3: Importer Données (3 min)
┌────────────────────────────────────────────────────────────────┐
│ Terminal → npm run simulation:import                           │
│ Attendre: ✅ IMPORT TERMINÉ (6 admins + 45 users + 300 sig)   │
└────────────────────────────────────────────────────────────────┘
